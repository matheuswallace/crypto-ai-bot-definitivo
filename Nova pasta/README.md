# Crypto AI Bot
Bot modular de inteligência artificial para análise de mercado de criptomoedas.

## Deploy no Railway
1. Faça upload deste projeto no GitHub.
2. Vá para [Railway.app](https://railway.app) e conecte o repositório.
3. Escolha a opção "Deploy from GitHub Repository".
4. O Railway detectará automaticamente o `docker-compose.yml` e iniciará os serviços.
